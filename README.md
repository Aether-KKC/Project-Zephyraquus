# Project Zephyraquus

Project Zephyraquus is an experimental engineering project focused on the design, prototyping, and documentation of an eight-motor distributed electric propulsion platform for studying flight control, power distribution, and autonomous systems.

## Safety

Project Zephyraquus is an experimental educational engineering project.

The current prototype is intended solely for laboratory experimentation and engineering learning. It is not designed or intended for human flight or transportation.
